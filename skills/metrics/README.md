# Metric Skills

Skills for configuring evaluation metrics.

## Available Skills

| Skill | Description |
|-------|-------------|
| [configure-metrics](./configure-metrics/) | Interactive guide to select and create evaluation metrics with use-case recommendations |
| [configure-trace-metrics](../traces/configure-trace-metrics/) | Create custom trace metrics from OpenTelemetry spans |

## Overview

Metrics define how agent performance is measured and scored. These skills help configure metric parameters and interpret evaluation results.

For metrics based on OpenTelemetry span names and attributes, use [configure-trace-metrics](../traces/configure-trace-metrics/).
