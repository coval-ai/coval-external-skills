# Reports

Skills for analyzing Coval reports and turning grouped evaluation results into concrete next steps.

| Skill | Description |
|-------|-------------|
| [analyze-audio-quality-report](./analyze-audio-quality-report/) | Analyze an audio-quality testing report, find scenario-specific regressions, inspect evidence, and recommend agent fixes |
| [analyze-accent-report](./analyze-accent-report/) | Analyze an accent testing report, find per-accent regressions (STT-first), inspect evidence, and recommend agent fixes |
| [analyze-adversarial-report](./analyze-adversarial-report/) | Analyze an adversarial / red-team report grouped by Test Case, find which attack vectors broke the agent, inspect evidence, and recommend prompt/guardrail/verification fixes |
