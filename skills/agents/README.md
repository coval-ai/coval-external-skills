# Agent Skills

Skills for creating and managing Coval agents.

## Available Skills

| Skill | Description |
|-------|-------------|
| [setup-agent](./setup-agent/) | Interactive guide to connect and configure a new agent for evaluation |
| [setup-tracing](../traces/setup-tracing/) | Add Coval OpenTelemetry tracing to an existing agent |

## Overview

Agents represent AI systems under evaluation. These skills help configure agent endpoints, authentication, and behavior settings.

For agents that already connect to Coval and need trace instrumentation, use the tracing skills under [Traces](../traces/).
